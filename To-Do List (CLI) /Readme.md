📝 To-Do List CLI App (Python)

A simple and beginner-friendly Command Line To-Do List application built using Python.
This project helps you manage daily tasks directly from the terminal.

---

🚀 Features

- ➕ Add new tasks
- 📋 View all tasks
- ❌ Delete tasks
- ✔️ Mark tasks as completed
- 💾 Save tasks using file handling

---

📁 Project Structure

todo_cli.py   # Main Python file
tasks.txt     # Stores tasks (auto-created)

---

▶️ How to Run

1. Make sure Python is installed
2. Clone the repository or download the file
3. Run the program:

python todo_cli.py

---

🧠 How It Works

- Tasks are stored in a list
- Data is saved in a file called "tasks.txt"
- The program runs in a loop until you exit
- Each menu option calls a specific function

---

📌 Menu Options

1. View Tasks
2. Add Task
3. Delete Task
4. Mark Task as Done
5. Exit
